📂 Nội dung được chia sẻ
⚙️ Dữ liệu được lưu trên Firebase. Bạn cần tự cấu hình Firebase và Security Rules (xem hướng dẫn trong code).
HƯỚNG DẪN: 1. Vào Firebase Console → Storage → Rules, đặt rules như sau: rules_version = '2'; service firebase.storage { match /b/{bucket}/o { match /shared/{fileId} { allow write: if request.auth != null; allow read: if request.auth != null && request.auth.token.email == resource.metadata.recipientEmail; } } } 2. Bật Authentication → Sign-in method → Email/Password. 3. Kiểm tra lại storageBucket: Vào Storage → xem tên bucket (thường là safe-data-vsip.appspot.com). Nếu không đúng, sửa lại trong firebaseConfig. 4. Mở file này trong trình duyệt (tốt nhất qua live server hoặc deploy).
